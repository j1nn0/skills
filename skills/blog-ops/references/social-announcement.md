# SNS 告知文

記事を SNS で告知するときは、記事の要約ではなく、読む理由をひとつだけ具体的に伝える。投稿・予約・送信は、ユーザーが明示的に依頼したときだけ行う。

## 書く順序

1. 記事の結論、失敗、比較結果、手順のいずれかひとつを最初に置く。
2. 誰が、どんな判断や作業に使えるのかを短く補う。
3. 記事 URL を添える。

記事のタイトル、summary、見出しをそのまま連結しない。本文にない成果、数字、感情を足さない。複数の話題やハッシュタグを詰め込み、記事への導線をぼかさない。

## 形式

```text
[記事で得られる具体的な結論または失敗]
[誰にどう役立つか]
[URL]
```

文字数やカード表示は投稿先と閲覧環境で変わるため、投稿前に実際の下書き表示で URL、改行、記事カードを確認する。ハッシュタグは投稿先の慣例と記事の内部タグを混同せず、必要な場合だけ付ける。

## 公開後の確認

- URL が意図した公開記事を開く。
- タイトル、summary、OGP 画像が記事の内容と一致する。
- 告知文が本文にない約束をしていない。
